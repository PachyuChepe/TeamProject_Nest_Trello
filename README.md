## Description

<img src="./public/image/logo.png" height="300"/>

### 칸반 보드

- **팀이 업무와 각 팀원이 할 수 있는 작업량 간에 밸런스를 맞추는 도구**입니다
  - 칸반 보드는 프로젝트의 업무량과 워크플로우를 더 효과적으로 시각화해 프로젝트를 관리합니다.
  - 보드에서 업무는 열로 구성된 프로젝트 보드에 표시되며, 각 열은 업무 단계를 나타냅니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
